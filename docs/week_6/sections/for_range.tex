\vspace{5mm}

The derivation tree will produce the result:
\begin{lstlisting}
    for x from 1 to 9
        y = 1
\end{lstlisting}
The purpose of the \emph{For Range} is to give the user the option to iterate via a variable. In this case, during the iterations, \emph{x} will take values from \emph{1} to \emph{9}, in this exact order.
If the range is set in a non-descending manner like \emph{from 6 to 1}, the compiler will identify that and adjust the iteration accordingly, meaning that during the iterations, \emph{x} will take values from \emph{6} to \emph{1}, in this exact order.
